# Track Fitness

Track Fitness is a simple web app for loggin your workouts and turning them into insights. Add any activity in seconds, see weekly and monthly trends, set goals that actually stick, and keep your profile dialed to your preferences. Clean, fast, and built to help you stay consistent. 

Core features 
- One-tap logging: Record date, activity type, duration
- Summary table: Filter and sort your history with totals at the top; edit or delete entries inline.
- Quick edits: Fix typos or update durations without leaving the page.

Enhanced features 
- Charts & trends: Weekly and monthly totals with bars/lines to show volume and streaks at a glance.
- Goal setting: Create weekly time goals or monthly distance targets; progress rings/bars track how close you are.
- User profiles: Name, avatar, units(mi/km), timeszone, default activity types, and data export.

Layout overview 
- Top Nav: Dashboard, Log Activity, History, Goals, Profile
- Dashboard (homepage)
  - Trend chart (weekly/monthly toggle)
  - Goal progress widgets
  - Quick Add button for fast logging
- Log Activity
  - Compact form (Date, Type, Duration, Distance, Notes)
  - Presets for common activities; submit adds to history and updates charts instantly
- History
  - Filter by date range/type; searching summary table(Date, Type, Duration, Distance,         Notes)
  - Row actions: Edit/Delete; optional bulk delete and CSV export
- Goals
  - Create/edit targets
  - Visual progess
- Profile
  - Personal details, units/timezone, preferred activities, data backup/export/import
