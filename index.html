Track Fitness

Web Development Project – Workout Tracker Dashboard

Track Fitness is a simple web app for logging workouts and turning them into insights. You can add activities in seconds, see weekly and monthly trends, set goals, and customize your profile.

Core Features

  • Log activities with date, type, duration, distance, and notes.
  • Dashboard with weekly trend bars, current streak, and goal progress.
  • History table with filters, search, and inline edit / delete.

Enhanced Features

  • Weekly & monthly goal tracking with progress bars.
  • Profile settings (name, units, timezone, default activities).
  • Simple login / create account using localStorage.

All layout, styling, and interactivity are implemented with HTML, CSS, and vanilla JavaScript. Data is stored locally in localStorage so activity history persists between page reloads.
